Пакет программ для проверки лабораторных работ вычислительного практикума

Демченко В.В., Симаков С.С. 

2007


Программный комплекс предназначен для использования при проверке правильности выполнения
заданий для самостоятельной работы, изложенных в учебном пособии: 
"Вычислительный практикум по прикладной математике", В.В. Демченко, М.: МФТИ, 2007.

Рекомендации по установке:
Программный комплекс специальной установки не требует. Однако, для нормальной работы 
программного комплекса может потребоваться Установка следующих компонент:
- .NET Framework 2.0 
(www.microsoft.com/downloads/details.aspx?displaylang=ru&FamilyID=0856EACB-4362-4B0D-8EDD-AAB15C5E04F5)

- Microsoft Windows Installer 3.0 
(www.microsoft.com/downloads/details.aspx?familyid=5FBC5470-B259-4733-A914-A956122E08E8&displaylang=en)
